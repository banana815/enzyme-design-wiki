# 学习路线图

酶设计与合成化学研究需要跨学科的知识体系。以下路线图按照"从基础到前沿、从实验到计算"的顺序组织。

## 1. 实验技术

### 酶学实验
- 酶的表达与纯化（质粒构建、转化、蛋白表达、柱层析）
- 酶活测定（初始速率法、稳态动力学、Michaelis-Menten 拟合）
- 定向进化技术（易错 PCR、饱和突变、筛选策略设计）
- 酶反应监测（LC-MS、GC-MS、NMR 实时监测）

### 合成化学实验
- 无水无氧操作（Schlenk 技术）
- 常见反应类型（Michael 加成、Aldol 反应、酰胺缩合）
- 产物分离纯化（柱层析、制备 HPLC、结晶）
- 结构鉴定（NMR 解谱、HRMS、IR）

## 2. 底物与路线设计——物理有机化学

### 核心知识
- 线性自由能关系（LFER）：Hammett 方程、Brønsted 催化方程
- 分子轨道理论：前线轨道（FMO）、Woodward-Hoffmann 规则
- 立体电子效应：异头效应、邻位孤对效应
- 反应机理实验方法：动力学同位素效应（KIE）、Hammett 图、停流实验

### 关键能力
- 从产物出发设计逆合成路线
- 判断关键步骤的可行性与选择性
- 利用 Reaxys/SciFinder 检索已知合成路线

## 3. 酶的应用——生物化学原理

### 核心知识
- 酶动力学：Michaelis-Menten、别构调控、过渡态理论
- 酶结构与功能：活性位点残基、催化三联体、辅因子
- 蛋白质工程：理性设计与定向进化的互补
- 代谢工程：酶在代谢通路中的语境

### 连接点
- 为什么有些反应在自由溶液中极慢但在酶活性位点中极快？（邻近效应、熵陷阱、共价催化、酸碱催化）
- 如何将这种理解迁移到非天然反应的设计中？

## 4. 语言学知识

### 一个出人意料的交叉

语言与化学之间存在深层类比：
- **语义压缩**：符号（如化学结构式）压缩了关于分子性质的丰富信息——就像语言中的词汇压缩了概念
- **深度神经网络的层数**：本质上是解压缩——逐层将压缩的表示展开为预测

参考：类似 Qwen 前负责人（自然语言处理背景）的研究思路——语言的语意压缩与表达如何启发化学表示学习。

### 需要了解的内容
- 信息论基础：熵、互信息、KL 散度
- 表示学习：Word2Vec → BERT → GPT 的演化逻辑
- 压缩与泛化的关系：MDL（最小描述长度）原则

## 5. 无标签学习问题

### 正无标签学习（PU Learning）

在酶反应数据中，我们通常只有"成功的反应"（正样本）和"未尝试的反应"（无标签样本）。负样本（失败的反应）往往不被记录。这使得传统的监督学习框架不直接适用。

- **间谍学习（Spy Learning）**：从无标签样本中抽取一部分作为"间谍"，假设其为负样本，训练分类器后检验间谍中被分类为正的比例，迭代调整
- **PU Learning**：直接利用正样本和无标签样本进行学习

## 7. 重整化群

### 物理概念向化学的迁移

重整化群是理论物理中处理多尺度问题的核心工具：

- **基本思想**：从微观的精细描述出发，逐层"粗粒化"，每一步丢掉不重要的细节，最终得到宏观的有效理论
- **与化学的关联**：
  - 分子的电子结构 → 官能团反应性 → 分子整体性质 → 复杂混合物的宏观行为
  - 每一层都在做"重整化"——用更少的参数描述更宏观的现象
  - 化学描述符设计本质上是一个寻找正确"粗粒化"参数的过程

这一视角可能为描述符的层级化设计提供新的思路——不是堆砌更多的描述符，而是寻找正确的"粗粒化层级"。

## 学习节奏建议

1. **第一阶段（0-6 个月）**：实验技术掌握 + 物理有机化学核心知识
2. **第二阶段（6-12 个月）**：酶学实验独立操作 + BO/迁移学习基础
3. **第三阶段（12-18 个月）**：独立设计底物和反应路线 + 基座模型微调实践
4. **第四阶段（18+ 个月）**：跨学科整合（语言学、重整化群等）+ PiEvo 框架实践
