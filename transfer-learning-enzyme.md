# 迁移学习在酶设计中的应用

## 核心问题

酶反应条件优化的一个根本困境：**每个新底物都需要重新做一轮条件筛选**。如果能将已有底物/条件下的知识迁移到新底物，实验成本将大幅降低。这就是迁移学习的用武之地。

## 三条迁移路径

### 路径 1：基于机理相似性的迁移（动力学方程拟合）

**前提**：底物之间的化学机理具有可迁移的共性。

方法：
1. 对已知底物体系拟合动力学常微分方程（ODE），获得各步骤的速率常数
2. 对于新底物，假设反应机理相同，但速率常数受底物电性/空阻影响发生系统性偏移
3. 基于 Hammett/Brønsted 等线性自由能关系（LFER），预测新底物的速率常数
4. 用少量实验验证和修正预测

**参考**：Nature 2023 年的常微分方程拟合工作——在机理明确时，ODE 拟合比纯黑箱模型更可靠、更可解释。

**挑战**：
- 需要补充较多的机理实验来建立 LFER 参数化
- 酶催化中的构象效应可能破坏简单的 LFER 关系
- 底物骨架变化较大时，机理相似性假设可能不成立

### 路径 2：传统机器学习小样本迁移

**前提**：不需要机理理解，纯粹从数据中学习底物-条件-产率的映射。

方法：
1. 在已有数据集（多个底物在不同条件下的产率）上训练 ML 模型
2. 将新底物的少量实验数据作为微调集
3. 使用元学习（meta-learning）或少样本学习（few-shot learning）框架适应新底物

**技术选项**：
- **MAML**（Model-Agnostic Meta-Learning）：训练一个能在少量梯度更新后快速适应新任务的初始模型
- **Prototypical Networks**：学习底物的嵌入表示，基于与已知底物的相似性进行条件推荐
- **Gaussian Processes with learned kernel**：将迁移知识编码在核函数中

**关键设计**：描述符必须跨底物骨架通用——电性（σ）和空阻（Es/Taft）参数天然具备这种普适性。

### 路径 3：大模型 + 小数据调参

**前提**：利用预训练大模型的化学先验知识，减少对酶反应特定数据的需求。

方法：
1. 使用化合物基础模型（如 ChemBERTa、MolFormer 等）提取底物的分子表示
2. 将反应条件（温度、溶剂、催化剂载量等）作为额外模态嵌入
3. 在小量酶反应数据上微调

**优势**：
- 大模型在预训练阶段已学习了大量化学结构-性质关系
- 酶反应微调只需要少量数据即可适配到特定体系
- 可以天然地融入文本知识（文献中的条件描述）

**开放问题**：
- 现有的化学基座模型主要在分子性质和小分子反应上预训练，对酶催化场景的覆盖有限
- 酶的序列/结构信息如何与底物表示融合？
- 微调数据量下限——一个底物/条件组合能否有效微调？

## 酶簇的普适性

一个更深层的问题：**不同突变体之间的迁移是否可能？**

酶簇（enzyme cluster）——同一酶的不同突变体——共享基本相同的催化机制和活性位点结构，但在底物偏好性和催化效率上存在差异。如果能在突变体之间进行迁移学习，将使酶的定向进化实验从"盲筛"转变为"预测导航"。

三个维度需要同时考量：
- 底物空间的迁移（同一酶，不同底物）
- 条件空间的迁移（同一酶+底物，不同条件）
- 突变体空间的迁移（同一底物+条件，不同突变体）

实际操作中，可以优先建立"底物空间迁移"的能力（最直接的需求），再逐步扩展到突变体空间。

## 与条件筛选的衔接

迁移学习不是条件筛选的替代，而是其**加速器**。在实际流程中：
1. 使用 BO 在少量代表性底物上做完整的条件优化（建立"锚点"）
2. 利用迁移学习将锚点知识扩展到新底物，给出初始条件建议
3. 在新底物上做小规模 BO 验证和修正
4. 新数据回流到迁移模型中，形成迭代改进的闭环
