# 反应条件筛选：动力学建模与贝叶斯优化

## 问题定义

酶催化反应的条件优化面临一个典型困境：**反应网络复杂、实验成本高、数据稀疏**。以 DERA-MA Michael 反应为例：

```
A + B → C + C'    （主反应与竞争副反应并行）
C → D             （主产物的后续分解）
```

宏观上表现为：存在**竞争的副反应** + **主产物的分解反应**。这意味着简单的"提高温度/延长反应时间 → 提高产率"的逻辑不适用——需要精确找到主反应速率、副反应速率和分解速率之间的最优平衡。

## 三条技术路线

### 路线 1：基于动力学方程的拟合

**适用条件**：机理明确，有足够机理实验支撑

1. 设计并执行机理实验，明确各步骤的速率方程参数
2. 基于动力学常微分方程（ODE）组，拟合实验数据给出最优条件组合
3. **关键要求**：参数必须有物理意义——这意味着每个拟合参数都需要独立的实验验证

**参考**：Nature 2023 年关于常微分方程拟合的工作提供了方法学框架——在已知机理的前提下，ODE 拟合可以给出比纯黑箱优化更可靠的预测。

**挑战**：
- 需要补充较多的机理实验（停流、同位素标记、中间体捕获等）
- 酶反应的中间步骤可能涉及构象变化，传统动力学方程难以捕捉

### 路线 2：贝叶斯优化（BO）

**适用条件**：机理不完全明确，以探索为主

BO 的关键设计选择：

- **边界条件设计**：如何定义反应条件的合理上下界？不改变现有的基本操作窗口
- **描述符连续性**：电性参数（如 Hammett σ）和空阻参数在底物空间中连续变化，但反应结果（产率）往往是离散化记录的
- **温度处理**：温度连续化处理——将离散的温度梯度转化为连续变量，纳入 BO 的采集函数
- **底物空间**：将肉桂醛与酮酸类底物共同放入化学空间，考虑电性影响

**0°C 产率异常**：一个实际观察是 0°C 反应产率反而较高。这提示反应可能受热力学和动力学双重控制——低温下副反应和分解反应被有效抑制。

**AI Skill 推荐**：已有研究者在使用 AI 推荐系统进行反应条件推荐。一个更务实的策略是将其作为辅助工具而非替代方案。

### 路线 3：保证多样性的 BO 优化

这是 BO 路线的一个进阶变体：

1. 定义底物空间（市售分子），通过描述符（电性 + 空阻，基于物化性质，确保可解释性）进行聚类采样
2. 使用描述符连续、结果离散化的 BO 框架
3. **核心创新**：在 BO 的采集函数中同时优化两个目标——**产率（最大化）** 和**与已测试点的相似性（限制）**

这样可以在探索新化学空间的同时避免陷入局部最优——既保证了底物多样性，又逐步提高产率。

## 底物空间定义

底物空间的定义直接影响优化的起点和效率：

1. **市售分子优先**：以可购买分子构建初始空间，降低合成压力
2. **聚类采样**：基于物化描述符（电性、空阻、logP、HOMO/LUMO 等）进行聚类，从每个簇中选取代表性分子
3. **描述符选择原则**：可解释性优先——选择有明确物理有机化学含义的描述符（Hammett σ、Taft Es、NBO 电荷等），避免纯计算指纹

## 酶-底物-条件交互空间

最具挑战性的问题是**酶簇的普适性**：不同突变体对同一底物/条件组合的响应可能截然不同。这引入了第三个维度：

```
底物空间 × 条件空间 × 酶突变体空间
```

完整探索这一三维空间是不可行的。可行的策略是：
- 先固定酶突变体，在底物×条件空间做 BO
- 识别出表现最优的少数底物/条件组合后，再横向测试不同突变体
- 利用迁移学习（见[迁移学习在酶设计中的应用](transfer-learning-enzyme.md)）在新突变体之间进行知识迁移

## 从筛选到优化

一个重要的区分：

- **筛选（Screening）**：从大量候选条件中找到有希望的起点——从 0 到 10
- **优化（Optimization）**：在已知有希望的区域内精调——从 10 到 90

两者需要不同的策略。筛选阶段需要广度（底物空间覆盖），优化阶段需要深度（对少数有希望条件的精细调节）。BO 框架可以同时服务这两个阶段，但采集函数的设计侧重点不同。
